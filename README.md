- Node.js (>= 18)

### Installation
Clone the repo:

bash
git clone git@github.com:AdityaMishra3000/artique.git
cd artique

Install dependencies (for both client & server):

cd client && npm install
cd ../server && npm install

Running Locally

Start frontend (React):

cd client
npm run dev

Start backend (Express):

cd server
npm run dev
